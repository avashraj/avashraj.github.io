Mesh analysis is a circuit analysis technique where we create fake mesh currents in closed loops and apply KVL. With these equations we can solve for whatever the problem asks for. Mesh analysis can only be done with planar circuits. To solve for a system of equations I started using the julia REPL. I have been wanting to learn julia for a few weeks and I finally found a way to use it.
